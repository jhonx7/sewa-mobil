## Sewa Mobil

Technical Test Jasamedika.

Tech Stack : Laravel, Inertia, React Js.

Jalankan perintah berikut untuk menjalankan aplikasi.
- `composer install`
- `npm install`
- `cp .env.example .env`
- `php artisan key:generate`
- `php artisan migrate` 
- `npm run dev` atau `npm run build`
- `php artisan serve`
